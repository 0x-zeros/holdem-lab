# holdem-lab

[https://0x-zeros.github.io/holdem-lab/](https://0x-zeros.github.io/holdem-lab/)

## 功能特性

- **胜率计算器**: Monte Carlo 模拟计算手牌胜率
- **手牌评估器**: 7 张牌选 5 张最优组合评估
